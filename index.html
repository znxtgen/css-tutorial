Shorthand properties

Useful for cleaner and shorter code, more readable.

Use:
border: 2px dashed orange
Instead of:
border-width: 2px
border-style: dashed | solid
border-color: orange

Also, for margin : top right bottom left or margin: 300px(top&bottom) 400px(left&right).

Box sizing

All elements by default happen to have a certain way of calculating width and height.

Content-box:

box-sizing: content-box

If we set width and height, we set them for the content, not the entire box including padding and border.

box-sizing: border-box

You want to use this one almost every time to target content + padding + margin.
